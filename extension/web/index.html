Why Lasso Copy

Built for structured content

Not just plain text. Lasso Copy captures the DOM — your tables stay as tables, your lists stay as lists.

Precision Selection

Draw any rectangle. Real-time highlighting shows exactly which elements will be captured using getBoundingClientRect() intersection detection.

Structured HTML Copy

Elements serialize as real HTML with text/html MIME type. Paste into Google Docs, Notion, Word — structure intact.

Images Preserved

Images, figures, and media are included in the selection. Computed styles are inlined so formatting travels with the content.

Instant Clipboard

Uses navigator.clipboard.write() with ClipboardItem for rich HTML. Falls back to execCommand for maximum compatibility.

Smart Deduplication

Ancestor-aware element filtering ensures clean output. Tables and lists are captured whole, not as fragmented pieces.

Zero Data Collected

Manifest V3 architecture. No servers. No analytics. Your clipboard is yours. Content never leaves your browser.
